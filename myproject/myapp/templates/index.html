Ejemplos propios del uso de If

Ejemplo 1: Numeros

    {% for number in numbers %}
  • {{ number }}
    • {% endfor %}

Ejemplo 2: Palabras

    {% for word in words %}
  • {{ word }}
    • {% endfor %}

Ejemplo 3: Usuarios

    {% for user in users %} {% if user.is_adult and show_adults %}
  • {{ user.name }} ({{ user.age }} )
    • {% endif %} {% endfor %}

Ejemplo 4: Fecha actual

Fecha de hoy: {{ current_date }}

Formato de fecha: {{ current_date|date:"F j, Y" }}

Ejemplo 5: Divisa

Cantidad: ${{ currency }}

Cantidad redondeada: ${{ currency|floatformat:2 }}
